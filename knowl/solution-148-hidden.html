Using the chain rule, we know that \(h'(x) = f'(g(x)) \cdot g'(x)\text{.}\) We use slope to compute the derivatives. Since \(f(x)\) is linear on the interval \(0 \lt x \lt 2\text{,}\) we compute the slope of the line to see that \(f'(x) = 1\) on this interval. Similarly, we compute the slope on the interval \(2 \lt x \lt 4\) to see that \(f'(x) = -1\text{.}\) Since \(f(x)\) has a corner at \(x = 2\text{,}\) we know that \(f'(2)\) does not exist.

Similarly, \(g(x)\) is linear on the interval shown, and we see that the slope of \(g(x)\) on this interval is \(-0.75\) so we have \(g'(x) = -0.75\text{.}\)

Then \(h'(1) = f'(g(1))\cdot g'(1) = f'(2.25) (-0.75) = (-1) (-0.75) = 0.75\)

\(h'(2) = f'(g(2))\cdot g'(2) = f'(1.5) (-0.75) = (1) (-0.75) = -0.75\)

\(h'(3) = f'(g(3))\cdot g'(3) = f'(0.75) (-0.75) = (1) (-0.75) = -0.75\)
