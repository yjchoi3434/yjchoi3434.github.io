Solution 5.3.5.2.1.
\(\int\,3 x^{5} \cos(x^{6})\,dx\text{.}\) We use substitution with \(w = x^{6}\) to find this. We have \(dw = 6\,x^{5}\,dx\text{,}\) so that
\begin{equation*} \int\,3 x^{5} \cos(x^{6})\,dx = {1\over 2}\int\,\cos(w)\,dw = {1\over 2}\,\sin(w) + C = {1\over 2}\,\sin(x^{6}) + C. \end{equation*}
