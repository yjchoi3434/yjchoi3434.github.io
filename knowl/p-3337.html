To determine which graphed function is the the derivative of another, we use the fact that the derivative is zero where the original function has a maximum or minimum. We note that A has zeros where C has a maximum or minimum, so it could be the derivative, so we can have \(A = C'\text{.}\) In addition, C has zeros where B has a maximum or minimum, so that it is possible that \(C = B'\text{.}\)
