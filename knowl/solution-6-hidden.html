SOLUTION a) The table gives the amount of calories used per minute. So in order to find the total number of calories used in a half-hour, use the table to find the number of calories used by a 150 lb who walks for a half-hour, and then multiply by 30 minutes. Using the table we know a 150 lb person walking uses 4 calories per minute; therefore,
\begin{equation*} \mbox{ Total calories used } = ( 4 )(30) = 120 \mbox{ calories} \end{equation*}
b) Using the same calculation as in (a), it follows that a 100 lb person swimming uses 5.8 calories per minute.
