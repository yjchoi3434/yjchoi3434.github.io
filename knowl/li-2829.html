Item f.

Knowing that average velocity is \(\frac{\pi}{2}\text{,}\) it follows from the fact that distance traveled equals average rate times time (provided velocity is always nonnegative), we have

\begin{equation*} D = \frac{\pi}{2} \cdot 4 = 2\pi\text{.} \end{equation*}

We see from (c) or (f) that we are simply considering the situation from two different perspectives: if we know the distance traveled, we can find average velocity, or if we know average velocity, we can find distance traveled.
