  3. Since we have the indeterminate form \(\frac{\infty}{\infty}\text{,}\) we can apply L'Hopital's Rule to determine the limit. In this case, we have

    \begin{equation*} \lim_{x \to \infty} \frac{\ln(x)}{x} = \lim_{x \to \infty} \frac{1}{x} = 0\text{.} \end{equation*}

    So we should expect that

    \begin{equation*} \lim_{n \to \infty} \frac{\ln(n)}{n} = \lim_{x \to \infty} \frac{\ln(x)}{x} = 0\text{.} \end{equation*}
