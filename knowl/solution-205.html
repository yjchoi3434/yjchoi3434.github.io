Solution 3.2.3.1.1.

SOLUTION

We have

\begin{equation*} T(D) = \left(\frac{C}{2} - \frac{D}{3}\right)D^2 = \frac{CD^2}{2}-\frac{D^3}{3}, \end{equation*}

and so

\begin{equation*} \frac{dT}{dD} = CD-D^2 = D(C-D). \end{equation*}

Since, by this formula, \(dT/dD\) is zero when \(D=0\) or \(D=C\text{,}\) negative when \(D>C\text{,}\) and positive when \(D\lt C\text{,}\) we have (by the first derivative test) that the temperature change is maximized when \(D=C\text{.}\)

The sensitivity is \(dT/dD = CD-D^2\text{;}\) its derivative is \(d^2T/dD^2 = C-2D\text{,}\) which is zero if \(D= C/2\text{,}\) negative if \(D> C/2\text{,}\) and positive if \(D\lt C/2\text{.}\) Thus by the first derivative test the sensitivity is maximized at \(D=C/2\text{.}\)
