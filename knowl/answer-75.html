Answer 2.4.2.1.

  1. All real numbers \(x\) such that \(x \ne \frac{\pi}{2} + k\pi\text{,}\) where \(k = \pm 1, \pm 2, \ldots\text{.}\)

  2. \(h'(x) = \frac{\sin(x)}{\cos^2(x)}\text{.}\)

  3. \(h'(x) = \sec(x) \tan(x)\text{.}\)

  4. \(h\) and \(h'\) have the same domain: all real numbers \(x\) such that \(x \ne \frac{\pi}{2}+k\pi\text{,}\) where \(k = \pm 1, \pm 2, \ldots\text{.}\)
