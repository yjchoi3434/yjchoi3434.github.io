The Second FTC provides us with a way to construct an antiderivative of any continuous function. In particular, if we are given a continuous function \(g\) and wish to find an antiderivative \(G\text{,}\) we can now say that

\begin{equation*} G(x) = \int_c^x g(t) \, dt \end{equation*}

provides the rule for such an antiderivative, and moreover that \(G(c) = 0\text{.}\) Note especially that we know that \(G'(x) = g(x)\text{,}\) or

\begin{equation} \frac{d}{dx} \left[ \int_c^x g(t) \, dt \right] = g(x)\text{.}\tag{5.2.2} \end{equation}
