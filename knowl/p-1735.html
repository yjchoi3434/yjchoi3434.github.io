  1. \(f(t) = \pi\) is constant, so \(f'(t) = 0\text{.}\)

  2. \(g(z) = 7^z\) is an exponential function, so \(g'(z) = 7^z \ln(7)\text{.}\)

  3. \(h(w) = w^{3/4}\) is a power function, thus \(h'(w) = \frac{3}{4} w^{-1/4}\text{.}\)

  4. \(p(x) = 3^{1/2}\) is constant, and therefore \(\frac{dp}{dx} = 0\text{.}\)

  5. \(r(t) = (\sqrt{2})^t\) is exponential (since \(\sqrt{2}\) is a constant), and so we have \(r'(t) = (\sqrt{2})^t \ln (\sqrt{2})\text{.}\)

  6. \(\frac{d}{dq}[q^{-1}] = -q^{-2}\text{,}\) by the rule for power functions.

  7. \(m(t) = \frac{1}{t^3} = t^{-3}\text{,}\) so \(\frac{dm}{dt} = -3t^{-4} = -\frac{3}{t^4}\text{.}\)
