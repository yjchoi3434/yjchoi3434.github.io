\begin{equation*} \mbox{average acceleration for } 0\le t\le 1 = \frac{\mbox{Change in velocity}}{\mbox{Change in time}}, \end{equation*}
so, reading the velocities from the table, we have
\begin{equation*} \frac{v(1) - v(0)}{1} = 33.41 - 0.00 = 33.41 \mbox{ ft/s}{}^2. \end{equation*}
Similarly,
\begin{equation*} \mbox{avg accel for } 1\le t\le 2 = \frac{v(2) - v(1)}{2-1} = 57.91 - 33.41 = 24.5 \mbox{ ft/s}{}^2. \end{equation*}
