Item 1.2.4.6.b.

Using a spreadsheet, we generate the following table.

\(x\) \(g(x)\)
\(-2.9\) \(-1\)
\(-2.99\) \(-1\)
\(-2.999\) \(-1\)
\(-3.001\) \(1\)
\(-3.01\) \(-1\)
\(-3.1\) \(-1\)

As we approach \(-3\) from the right, the function values approach (stay constant at) \(1\text{;}\) but as we approach \(-3\) from the left, the function values approach (stay constant at) \(-1\text{.}\) As such, the function values do not get arbitrarily close to a single value, and hence the limit does not exist.
