\(H(x) = F(G(x))\text{,}\) \(H(1) = F(G(1)) = F(3) = 7\)\(H'(1) = F'(G(1))\cdot G'(1) = F'(3)\cdot 1 = 2\cdot 1 = 2\) For \(H(x) = G(F(x))\text{,}\) \(H(1) = G(F(1)) = G(2) = 7\)\(H'(1) = G'(F(1))\cdot F'(1) = G'(2)\cdot 2 = 5\cdot 2 = 10\) For \(H(x) = F(x)/G(x)\text{,}\) \(H'(1) = \frac{F'(1) G(1) - F(1) G'(1)}{G(1)^2} = \frac{ (2)(3) - (2)(1) } {3^2} \approx 0.444\)
