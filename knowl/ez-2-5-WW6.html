Exercise 6. Chain rule with function values.

Given \(F(1)=2, F'(1)=2, F(3)=7, F'(3)=2\) and \(G(1)=3, G'(1)=1, G(2)=7, G'(2)=5\text{,}\) find each of the following. (Enter dne for any derivative that cannot be computed from this information alone.)

  1. \(H(1)\) if \(H(x)=F(G(x))\)

  2. \(H'(1)\) if \(H(x)=F(G(x))\)

  3. \(H(1)\) if \(H(x)=G(F(x))\)

  4. \(H'(1)\) if \(H(x)=G(F(x))\)

  5. \(H'(1)\) if \(H(x)=F(x)/G(x)\)
