\begin{equation*} \frac{3 x + 3}{x^2-3x+2} = \frac{3 x+3}{(x-1)(x-2)} = \frac{A}{x-1} + \frac{B}{x-2}. \end{equation*}
Multiplying by \((x-1)\) and \((x-2)\text{,}\) this becomes
\begin{equation*} 3 x+3 = A(x-2)+B(x-1), \end{equation*}
or
\begin{equation*} A+B = 3 \quad\hbox{and}\quad -2A - B = 3. \end{equation*}
Solving this system yields \(A = -6\) and \(B = 9\text{.}\) Then
\begin{equation*} \int\frac{3 x+3}{x^2-3x+2}dx = \int\left(\frac{9}{x-2} - \frac{6}{x-1}\right)\,dx = 9\ln\!\left(\left|x-2\right|\right)-6\ln\!\left(\left|x-1\right|\right)+C. \end{equation*}
