Item 4.3.5.7.e.

Assuming that \(v\) is linear on the interval \([0,1.5]\text{,}\) the slope of \(v\) is \(-\frac{3}{1.5} = -2\text{.}\) Given that the \(v\)-intercept is \((0,1)\) we conclude that on the interval \([0,1.5]\) we have \(v(t) = -2t+1\text{.}\) Now position, \(s\text{,}\) is an antiderivative of the velocity, \(v\text{,}\) so the possible formulas for \(s\) on the interval \([0,1.5]\) are \(s(t) = -t^2+t+C\text{,}\) where \(C\) is some constant. If we want \(s(0) = 0\text{,}\) that forces \(C=0\) and so the object's position function on \([0, 1.5]\) that satisfies \(s(0) = 0\) is \(s(t) = -t^2+t\text{.}\)
